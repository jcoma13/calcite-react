# ArcgisItemCard

A card for ArcGIS items to be displayed as a single UI element. 
Use the playground to adjust the features displayed on the card.

#### Import Syntax

```js
import ArcgisItemCard from 'calcite-react/ArcgisItemCard'
```

## Basic Usage

<Playground>
  <ArcgisItemCard 
    item={item} 
    actions={[<StyledUserIcon/>, <StyledCalendarIcon/>]} 
    selectable 
  />
</Playground>

## Vertical Usage

<Playground>
  <ArcgisItemCard 
    item={item} 
    vertical 
    actions={[<StyledUserIcon/>, <StyledCalendarIcon/>]} 
    selectable 
  />
